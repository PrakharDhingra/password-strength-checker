# 🔐 Password Strength Checker

## 📌 About
An AI-powered password strength checker and generator built with Python.


## ❗ Problem Statement
People in our community often use weak passwords for online accounts,
leading to increased risks of hacking, identity theft and data breaches.

---

## ✅ Features
- Checks password strength: Weak / Medium / Strong / Very Strong
- Gives specific improvement suggestions
- Generates strong random passwords automatically
- Color-coded terminal output
- Visual strength bar display

---

## ▶️ How to Run
```bash
python password_checker.py
```

---

## 🛠️ Tech Used
- Python 3
- Libraries: `random`, `string` (both built-in)

---

## 📊 How It Works
1. User opens the application
2. Enters their password OR chooses to generate one
3. AI evaluates strength based on:
   - Length (8+ or 12+ characters)
   - Uppercase letters
   - Lowercase letters
   - Digits
   - Special characters
4. Gets a strength score and improvement tips
